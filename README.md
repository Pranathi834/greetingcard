# Greeting Card App

This is a simple Android application developed using Kotlin and Jetpack Compose.

## Features
- Displays a greeting message.
- Custom name display.
- Built using Android Studio.

## Output
The app shows: "Hi, my name is Pranathi!"

## Tools Used
- Android Studio
- Kotlin
- Jetpack Compose
- Emulator (Pixel 6 API 34)
